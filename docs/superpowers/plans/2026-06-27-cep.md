# Busca por CEP — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Um campo de CEP no cadastro/edição que consulta o ViaCEP e preenche rua/bairro/cidade/estado, guardando o CEP no venue.

**Architecture:** Coluna `cep` no `venues`. Componente `CepInput` (criar+editar) que, ao completar 8 dígitos, busca no ViaCEP e devolve os campos via callback. Backend só persiste o `cep` (autofill é frontend).

**Tech Stack:** Go + Gin, pgx/sqlc, Next.js 15 + React 19 + TS, ViaCEP.

## Global Constraints
- **Coluna `cep`** TEXT NOT NULL DEFAULT '' (só dígitos). Migration initdb → aplicar manual no DB de QA.
- **ViaCEP** `https://viacep.com.br/ws/<8digits>/json/`; mapeia `logradouro→address`, `bairro→neighborhood`, `localidade→city`, `uf→state`. **Não** mexe em complemento/número.
- **Auto-busca ao completar 8 dígitos** (sem botão). CEP opcional; falha/não-encontrado → preenche manual.
- **Gates:** `cd backend && go test ./... && go build ./...`; `cd frontend && npm run typecheck && npm run build`.

## Task 4: Componente `CepInput`

**Files:**
- Create: `frontend/app/venues/cep-input.tsx`

**Interfaces:**
- Produces: `default CepInput({ cep, onCepChange, onResolve }: { cep: string; onCepChange: (cep: string) => void; onResolve: (r: { address: string; neighborhood: string; city: string; state: string }) => void })`.

- [ ] **Step 1: Criar o componente**

```tsx
'use client';

import { useState } from 'react';

interface Resolved {
  address: string;
  neighborhood: string;
  city: string;
  state: string;
}

interface CepInputProps {
  cep: string;
  onCepChange: (cep: string) => void;
  onResolve: (r: Resolved) => void;
}

export default function CepInput({ cep, onCepChange, onResolve }: CepInputProps) {
  const [status, setStatus] = useState('');

  async function onChange(raw: string) {
    const digits = raw.replace(/\D/g, '').slice(0, 8);
    onCepChange(digits);
    if (digits.length !== 8) {
      setStatus('');
      return;
    }
    setStatus('Buscando endereço…');
    try {
      const res = await fetch(`https://viacep.com.br/ws/${digits}/json/`);
      const d = await res.json();
      if (d.erro) {
        setStatus('CEP não encontrado — preencha manualmente.');
        return;
      }
      onResolve({
        address: d.logradouro || '',
        neighborhood: d.bairro || '',
        city: d.localidade || '',
        state: d.uf || '',
      });
      setStatus('Endereço preenchido — confira e complete o número.');
    } catch {
      setStatus('Não consegui buscar o CEP — preencha manualmente.');
    }
  }

  return (
    <label>
      CEP
      <input
        value={cep}
        inputMode="numeric"
        maxLength={8}
        placeholder="Só números (8 dígitos)"
        onChange={(e) => onChange(e.target.value)}
      />
      {status && <span className="muted">{status}</span>}
    </label>
  );
}
```
